This is a basic motor control using a arduino code with keypad control and a lcd 16*2 display.

Major components used are,
1. Arduino Uno
2. Motor with encoder (model: rmcs-5033)
3. Motor driver (model: rmcs - 2303)
4. Keypad (4*4)
5. LCD display (16*2)
6. Power source 12V ( to power the driver and motor)
